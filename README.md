##  How It Works

1. The game generates a secret number based on the current level.
2. Player enters a guess.
3. The game responds:
   - Too Low
   - Too High
   - Correct
4. Attempts decrease after each guess.
5. A hint reveals whether the number is EVEN or ODD.
6. Correct guess advances to the next level.
7. Game ends when:
   - Attempts reach zero
   - All 10 levels are completed
